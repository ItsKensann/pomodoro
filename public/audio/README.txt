Drop this optional audio file in this folder for local sound:

  chime.mp3       - short chime played when a session ends

Suggested royalty-free sources:
  - pixabay.com/sound-effects/  (search "soft chime")
  - freesound.org
  - zapsplat.com

The app will silently skip chime playback if the file is missing.
Lofi music streams through YouTube using the in-page lofi.radio player.
