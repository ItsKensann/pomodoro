The app ships with a small default chime:

  chime.wav       - short chime played when a session ends

You can replace it with another short WAV file using the same filename.

Suggested royalty-free sources for replacement sounds:
  - pixabay.com/sound-effects/  (search "soft chime")
  - freesound.org
  - zapsplat.com

The app will silently skip chime playback if the file is missing or blocked by the browser.
